# 竹溪社照片遮脸工作台

打开 `index.html` 即可使用。建议用 Chrome 或 Edge。

## 素材库

- 方块像素头像：`assets/faces/`
- 粉笔质感简笔画：`assets/chalk-doodles/`
- 素材清单：`assets/manifest.json`
- 预览图：`asset-library-preview.jpg`

## 使用顺序

1. 点击“选择照片”，可一次选多张。
2. 在右侧选择“头像”或“粉笔画”素材。
3. 点击照片添加素材。
4. 拖动、滚轮缩放，或用右侧滑杆调整大小和旋转。
5. 导出当前 PNG，或用 Chrome/Edge 批量导出到文件夹。

## 注意

- 全部处理都在本地浏览器完成，不上传照片。
- 导出的标注 JSON 只保存贴纸位置，不包含照片本体。
- 再次导入标注前，需要先选择同名照片。
